# Using the Xaman — XRPL Manager Chrome Extension

This repository provides notes, tips, and example workflows for the **Xaman — XRPL Manager** Chrome extension – a browser extension that helps you interact with the XRP Ledger (XRPL) directly from your browser.

> **Note:** This project is **unofficial** and **not affiliated** with Xaman or Ripple. It’s just a helper/guide for users who enjoy the Xaman — XRPL Manager Chrome extension.

---

## 1. What is the Xaman — XRPL Manager Chrome Extension?

The **xaman chrome extension** is designed to connect your browser to your XRP/XRPL experience. It works together with your Xaman mobile app, letting you:

- View and sign XRPL transactions from your browser  
- Interact with XRPL-enabled websites and dApps  
- Keep sensitive keys safely on your phone while using the desktop browser

You can find the official listing here:

- **Official Chrome Web Store URL:**  
  `https://chromewebstore.google.com/detail/xaman-%E2%80%94-xrpl-manager/hbaamilhgknggfanbiceijijemigbbah`

Always make sure you install from this exact link:
